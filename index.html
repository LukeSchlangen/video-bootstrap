About Video Bootstrap

Video Bootstrap is a free Bootstrap 3 theme created by Luke Schlangen and based on the Grayscale Theme from Start Bootstrap and the youtube player from pupunzi. This theme features video from Max Schlickenmeyer along with this picture from Casey Fyfe from unsplash.com. The theme is open source, and you can use it for any purpose, personal or commercial.

Video Bootstrap includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization and can be downloaded below.

Video Bootstrap Feature Demo

This theme allows you to use bootstrap modals for displaying youtube videos, and unlike default bootstrap, will stop the youtube video when the modal is closed. These videos can be triggered the same way as a bootstrap modal. Here are a few examples of what it might look like.
