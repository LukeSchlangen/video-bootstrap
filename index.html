Video Bootstrap

About Video Bootstrap

Video Bootstrap is a free Bootstrap 3 theme created by Luke Schlangen and based on the Grayscale Theme from Start Bootstrap and the youtube player from pupunzi. This theme features video from Max Schlickenmeyer along with this picture from Casey Fyfe from unsplash.com. The theme is open source, and you can use it for any purpose, personal or commercial.

Video Bootstrap includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization and can be downloaded from github.

Demo

This theme allows you to use bootstrap modals for displaying youtube videos, and unlike default bootstrap, will stop the youtube video when the modal is closed. These videos can be triggered the same way as a bootstrap modal. Here are a few examples of what it might look like.

Help

There are a few questions I'll try to help with from the start.

How do I use the youtube videos I want?

I moved all of the image links and video references out of the javascript and css and into the html for ease of use. You just need to change the youtube links to the videos you want, and you'll be all set. If you want some help figuring out the youtube api, this is an incredible resource.

Why isn't it automatically playing on phones?

Most modern browsers won't auto-start videos on mobile devices because you could eat up users' data on accident. Instead of leaving a picture of a youtube video, I just have the original background image stay put.

Why does this mute all youtube videos?

It's because of the way youtube works and what the YTPlayer (documentation here) does in order to make it so that audio doesn't start playing right away. To change it, there is a part in the html that says "mute:true" and you can set that to "mute:false".

The problem is that the audio will now start playing right away, and there isn't an easy way for the user to turn it off (annoying). The best way to solve this, is to use a video from youtube that doesn't use audio (is silent) and then set "mute:false". Then you don't have to mute everything, and you don't have audio start right away.
